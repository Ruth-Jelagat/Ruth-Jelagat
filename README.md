## Running the Project Locally

### 1. Clone the repository

```bash
git clone git@github.com:Ruth-Jelagat/My-Portfolio.git
cd My-Portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start development server

```bash
npm run dev
```

### 4. Open in browser

```
http://localhost:3000
```
